# Terminal CV

Terminal-style portfolio website built with Flask.

## Features

- Terminal-inspired interface
- AWS theme colors
- Responsive design
- Project showcase
- Contact form
- Work experience timeline

## Tech Stack

- **Backend**: Flask (Python)
- **Frontend**: HTML5, CSS3, JavaScript
- **Fonts**: Fira Mono
- **Icons**: Font Awesome

## Setup

```bash
# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run application
python app.py
```

Visit: http://localhost:5051

## Customization

- Update personal information in templates
- Add your photo as `static/profile.jpg`
- Modify projects in `templates/ls.html`
- Update work experience in `templates/work.html`
